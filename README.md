# vue-project

### Set up ICE server
```
docker pull coturn/coturn:latest
docker run -d --network host coturn/coturn:latest
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
